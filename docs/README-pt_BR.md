## Descrição do Projeto ##

Bem vindos ao Race Chrome, um tema base desenvolvido pelo [Grupo WordPress Brasil](https://www.facebook.com/groups/wordpress.brasil), para auxiliar no desenvolvimento ágil de temas para WordPress.

## Instalação ##

Envie a pasta "race-chrome" para a pasta de temas padrão do WordPress `wp-content/themes` ou simplesmente envie o arquivo zipado utilizando o instalador de temas do WordPress.

## Documentação ##

Informações de como usar as classes e funções do Race Chrome na [Wiki Oficial](https://github.com/wpbrasil/race-chrome/wiki/)

## Suporte a plugins ##

Race Chrome suporta nativamente os plugins:

* [WooCommerce](http://wordpress.org/extend/plugins/woocommerce/)
* [JetPack](http://wordpress.org/extend/plugins/jetpack/) Infinite Scroll
